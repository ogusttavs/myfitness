# Prints

Pasta pra colocar screenshots/imagens que você quer que o Claude veja.

## Como usar

1. Cola o print aqui (drag, ou save_as do navegador, ou screenshot direto)
2. Na conversa, fala "olha o print em prints/{nome-do-arquivo}" ou só "tem print novo em prints/"
3. Eu leio com a tool `Read` e vejo a imagem

## Convenções

- Nomes descritivos: `erro-login-coach.png`, `dashboard-vitor.jpeg`, etc — facilita pra você lembrar
- Limpa o que não precisar mais — não tem auto-clean
- Tudo aqui está gitignored (não vai pro repo)

## Dev Notes
- Cliente faz warn em dev se vars faltam — não crasha pra permitir UI dev sem backend.
- Quando tiver projeto Supabase: rodar `npx supabase gen types typescript` e substituir database.types.ts.
- Auth persistido em MMKV via adapter síncrono; QueryClient persister usa wrapper async (TanStack exige Promise).
- `react-native-get-random-values` shim necessário pra crypto.getRandomValues do uuid.
- `react-native-url-polyfill/auto` necessário pra Supabase em ambiente RN.
